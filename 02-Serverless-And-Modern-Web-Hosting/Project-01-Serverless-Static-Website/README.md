# Project 01: Serverless Static Website Hosting on AWS

This project demonstrates hosting a fully serverless static website using Amazon S3 and Amazon CloudFront for global content delivery.

## Features
- Serverless static website hosting
- Global content delivery using CloudFront CDN
- Secure access to S3 using CloudFront
- HTTPS-enabled website
- No server or infrastructure management

## AWS Services Used
- Amazon S3
- Amazon CloudFront
