# Serverless Application & Modern Web Hosting Projects

This section contains AWS projects focused on **serverless architectures and modern web hosting approaches**, leveraging fully managed services to build scalable, secure, and cost-efficient applications without managing servers.

---

## Projects

1. **Serverless Static Website**  
   Hosted a static website using Amazon S3 and CloudFront with edge-level optimization for global performance.

2. **Multi-Tenant SaaS Application**  
   Designed a multi-tenant SaaS architecture using Amazon Cognito, API Gateway, and Amazon RDS.

3. **Progressive Web App (PWA) Hosting**  
   Deployed a React/Vue Progressive Web App using AWS Amplify with CI/CD and HTTPS.

4. **Jamstack Website with AWS**  
   Built a Jamstack site using AWS Amplify, a headless CMS, and Amazon S3 for fast and secure content delivery.

5. **AWS App Runner Deployment**  
   Deployed a containerized web application using AWS App Runner with automatic scaling.

6. **Serverless REST API**  
   Built a RESTful API using AWS Lambda and API Gateway without managing servers.

7. **GraphQL API with AWS AppSync**  
   Implemented a GraphQL backend using AWS AppSync and Amazon DynamoDB.

8. **Serverless Chatbot**  
   Created an AI-powered chatbot using Amazon Lex and AWS Lambda.

9. **Event-Driven Microservices**  
   Designed asynchronous microservices communication using Amazon SNS and Amazon SQS.

10. **URL Shortener**  
    Built a serverless URL shortener using AWS Lambda and Amazon DynamoDB.

11. **Serverless Expense Tracker**  
    Developed an expense tracking application using Lambda, DynamoDB, and API Gateway.

12. **Serverless Image Resizer**  
    Automatically resized images uploaded to Amazon S3 using Lambda triggers.

13. **Real-Time Polling Application**  
    Built a real-time polling system using WebSockets with API Gateway and DynamoDB.

14. **Serverless Sentiment Analysis**  
    Analyzed tweet sentiment using AWS Comprehend and stored results in DynamoDB.

---

## Key Concepts Covered
- Serverless architecture
- Event-driven systems
- REST and GraphQL APIs
- Authentication and authorization
- CDN and edge computing
- Real-time communication
- Scalable cloud-native application design
