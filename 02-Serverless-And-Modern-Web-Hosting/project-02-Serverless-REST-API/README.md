Project 02: Serverless API using AWS Lambda & API Gateway

This project demonstrates building a fully serverless backend API using AWS Lambda and Amazon API Gateway to handle HTTP requests and return JSON responses.

Features

Fully serverless REST API

Public HTTP endpoint using API Gateway

Backend logic executed using AWS Lambda

JSON-based API responses

Automatic scaling and high availability

No server or infrastructure management

AWS Services Used

AWS Lambda

Amazon API Gateway (HTTP API)

AWS IAM
