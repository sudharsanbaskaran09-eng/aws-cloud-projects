# Project 03: Static Website Hosting on AWS EC2 (Apache)

## Overview
This project demonstrates deploying a static website on an Amazon EC2 instance using the Apache web server.

## Services Used
- Amazon EC2
- Amazon Linux 2023
- Apache (httpd)

## Steps Performed
- Launched EC2 instance
- Configured security groups (SSH & HTTP)
- Installed and started Apache
- Deployed custom HTML website
