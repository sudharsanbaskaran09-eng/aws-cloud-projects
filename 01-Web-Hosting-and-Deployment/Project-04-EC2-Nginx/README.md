# Project 4: Static Website Hosting on AWS EC2 using Nginx

## Overview
This project demonstrates hosting a static website on an Amazon EC2 instance using the Nginx web server.

The server is configured on Amazon Linux and serves a custom HTML page via the public IPv4 address.

## Services Used
- Amazon EC2
- Amazon Linux
- Nginx Web Server
- AWS Security Groups

## Steps Performed
1. Launched an EC2 instance
2. Configured security group to allow HTTP (port 80)
3. Installed Nginx web server
4. Deployed static HTML content
5. Verified access via public IP

## Outcome
The website is successfully hosted and accessible through the EC2 public IPv4 address.
