# Project 02: WordPress Deployment on AWS Lightsail

This project demonstrates deploying a WordPress website on AWS Lightsail
using the Bitnami stack.

## Features
- Linux-based Lightsail instance
- WordPress installed using Bitnami
- SSH access for server management
- Static public IP configured
- WordPress admin dashboard access

## Access
- Public Site: http://3.6.20.192/wp-admin
- Admin Panel: wp-admin (restricted access)

## AWS Services Used
- Amazon Lightsail
